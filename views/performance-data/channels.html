Guidance Transactions and channels

User effort is a way to measure the different reasons services might be creating extra effort for their users. This page contains further detail on what the different categories include.

Reasons users get in touch with services

I need more information

This could include:

  • couldn't find what they needed online
  • couldn't understand our website
  • needs to know more about something

I want to check the status of something

This could include:

  • wants to know what's happened to information they've previously supplied
  • wants to know if something is still valid
  • wants to know why they haven't heard back about something

I want to challenge a decision

This could include:

  • wants to appeal a decision
  • has been sent incorrect information

I need additional support

This could include:

  • is unable to use the service due to a disability
  • isn't confident using a computer
  • doesn't have access to a computer

I had to get in contact

This could include:

  • was asked to get in contact as part of the service
  • contacting the service is the only way to do what they want to do

Reasons users have to repeat or amend something

User's information is illegible

This could include:

  • user's handwriting is illegible
  • user has made typo errors

User's information is incomplete

This could include:

  • user has omitted key information
  • user has submitted with wrong or missing supporting information
  • there is no wet signature
  • government handling errors have resulted in missing information

User completed the wrong thing

This could include:

  • user has applied for the wrong thing
  • the wrong form (for example, an outdated or wrong version) has been used
  • the user has attempted to complete several applications on one form

User's information doesn't match government's

This could include:

  • the data submitted by the user does not match government records
  • user provided false information
